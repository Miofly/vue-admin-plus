## 安装使用

- 获取项目代码

`github` 方式(保持最新代码)

```bash
git clone https://github.com/Miofly/vue-admin-plus.git
```

`gitee` 方式(最新代码更新时间不定)

```bash
git clone https://gitee.com/miofly/vue-admin-plus
```

- 安装依赖

```bash
cd vue-vben-admin

pnpm install

```

- 运行

```bash
pnpm serve
```

- 打包

```bash
pnpm build
```
