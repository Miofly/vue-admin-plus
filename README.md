## Install and use

get the project code

- github method

```bash
git clone https://github.com/Miofly/vue-admin-plus.git
```

- gitee method(code maybe old)

```bash
git clone https://gitee.com/miofly/vue-admin-plus.git
```

install

```bash
cd vue-admin-plus

pnpm install

```

run

```bash
pnpm serve
```

build

```bash
pnpm build
```
